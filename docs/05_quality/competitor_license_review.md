## 1. 목적·범위

경쟁 앱 **공개 웹·앱 데이터**를 §30-4-4 Competitor Feed(Layer 3-C)로 통합하기 전, **robots.txt**, **이용약관(ToS)**, **공개 형식(웹/API)** 관점에서 재사용 가능성을 **GREEN / YELLOW / RED / UNKNOWN**으로 분류한다.

대상:

| 구분 | 서비스 | 조사에 사용한 대표 URL·출처 |
|------|--------|---------------------------|
| 대한민국 | 더콜 (TheCall) | `https://www.thecall.co.kr/` · `https://www.thecall.co.kr/robots.txt` |
| 대한민국 | 후후 (WhoWho) | `https://whowho.vp.co.kr/ko/` · `https://whowho.vp.co.kr/ko/terms/` · `/robots.txt` (404) |
| 대한민국 | 뭐야이번호 | `https://moyaweb.com/` · `http://idc-kr-notice.whycall.net/WHO2/agree/agree1.htm` (이용약관 링크) |
| 대만 | Whoscall | `https://whoscall.com/` 계열 · Gogolook `https://www.gogolook.com/terms-of-use` · 이용약관 조항 번역본 |

**면책**: 본 보고서는 법률 자문이 아니다. 실제 통합·크롤링·API 연동 전에는 **법무 또는 라이선스 담당 검토**가 필요하다.

---

## 2. 분류 정의

| 등급 | 의미 | 제품 조치(권고) |
|------|------|----------------|
| **GREEN** | 공식 파트너십·계약 API·명시적 재사용 허용 등 **통합 근거 문서상 명확** | 계약 범위 내 구현 가능 |
| **YELLOW** | robots 등 **일부 신호는 허용**이나 ToS상 **자동화·재배포·2차 서비스 금지** 가능성이 큼; 또는 **원문 미확보**로 완전 평가 불가 | **상업적 라이선스 협상** 또는 법무 검토 후에만 설계 |
| **RED** | 이용약관 등에서 **무단 수집·봇·역설계·영리 무단 이용** 등이 명확히 금지되거나, 일반 경쟁사 DB 재판매와 양립 어려움 | **무허가 크롤링·벌크 미러링 금지**. 공식 채널만 검토 |
| **UNKNOWN** | **robots.txt / 약관 본문 확보 실패** 또는 해석 불충분 | 기술 재시도 + 법무·현지 법 근거 확보 전까지 **구현 보류** |

## 3. 서비스별 검토

### 3.1 더콜 (TheCall, 한국)

**robots.txt** (조사 시 `curl`, 브라우저 UA): `User-agent: *` / `Disallow:` 비어 있음 → **크롤러 차단 디렉터리 미설정**(프로토콜 관점).

**ToS**: 본 라운드에서 **전문 다운로드·인용 확보 없음**(사이트 탐색 필요). robots만으로 **데이터베이스·저작권 재사용 권한이 발생하지 않음**.

**공개 형식**: 번호별 검색 결과·게시판 형태의 웹 UI가 중심일 가능성이 높음. **벌크 오픈 데이터 포맷/API 공개 여부는 미확인**.

| 항목 | 판정 |
|------|------|
| robots.txt | 프로토콜상 제한 약함 → 단순 **크롤 허용 신호** 수준 |
| ToS | **UNKNOWN** (본문 미첨부) |
| **종합** | **YELLOW** — robots만 보고 GREEN 불가. 무허가 벌크 재판매는 피하고, **UNKNOWN 해소·법무 검토** 선행 권고 |

---

### 3.2 후후 (WhoWho, 한국 · KT 계열)

**robots.txt**: `https://whowho.vp.co.kr/robots.txt` → **HTTP 404** (파일 부재). 검색엔진 정책은 **UNKNOWN**.

**ToS**: 공식 경로 `https://whowho.vp.co.kr/ko/terms/` 존재하나, 본 조사에서는 **전문 텍스트 추출 전 UI 골격 위주**만 확보. 통신·전화번호부 연계 서비스 특성상 **DB 무단 재가공 금지 조항이 있을 가능성이 높음**(일반 관행적 추정 — 법적 확정 아님).

| 항목 | 판정 |
|------|------|
| robots.txt | **UNKNOWN** (404) |
| ToS | **UNKNOWN** (본문 미인용) |
| **종합** | **YELLOW** (보수적 기본값). 제3자 경쟁 앱의 자동 다운로드 소스로 쓰기 전 **공식 계약 경로** 확인 필요 |

---

### 3.3 뭐야이번호 (에바인, 한국)

**robots.txt** (`https://moyaweb.com/robots.txt`): `User-agent: *` · `allow: /` → **경로 전체 허용 형태**(표기 단순).

**이용약관** (`agree1.htm`, 회사: 에바인): 제11조 회원 의무에서 다음과 같이 규정됨(요지):

- 다른 회원의 **개인정보·계정정보 수집 금지**
- **회사 사전 동의 없는 영리 목적** 이용 금지
- **리버스 엔지니어링**, 디컴파일 등으로 서비스 **복제·모방·변형 금지**
- **자동 접속 프로그램** 등 정상적이지 않은 방법으로 서비스 이용하여 **서버 부하·방해 금지**
- 회사 동의 없이 **영리 목적**으로 서비스 사용 금지

→ 공개 웹 조회를 경쟁사 피드로 **벌크 저장**하는 시나리오는 **회사 동의·영리 이용·자동화**와 충돌 소지가 크다.

| 항목 | 판정 |
|------|------|
| robots.txt | 형식상 허용 → **라이선스 ≠ GREEN** |
| ToS | 자동화·영리 무단·복제 금지 근거 **명확** |
| **종합** | **YELLOW ~ RED** (무허가 스크래핑·경쟁 제품용 재판매는 **RED**로 간주). 공식 협의 없는 통합 **비권고** |

---

### 3.4 Whoscall (走着瞧 / Gogolook, 대만)

**robots.txt**: `https://whoscall.com/robots.txt` 등 다수 경로에서 **302·404** 등으로 **일관된 텍스트 미확보**(CDN·리전별 분기 가능). → **UNKNOWN**(프로토콜).

**Gogolook 기업 사이트** `https://www.gogolook.com/robots.txt`: `User-agent: *` / `Allow: /` — **기업 소개 사이트**에 대한 신호이며 Whoscall 앱 DB와 동일하지 않음.

**Whoscall 이용약관**(발효일 2025-06-26 조항, 영한 대역 텍스트 확본본 기준 요지):

- 제7조 **금지 행위**: 무단으로 시스템에 **침입·간섭**, 보안 **취약점 스캔**, **소프트웨어 역설계**, **봇 등 자동화 수단**으로 서비스 내 데이터·내용 **조작·파괴**, 앱 **무단 수정·재배포·제3자 제공** 등 금지
- 제8조 **라이선스**: **비독점·양도 불가·재허가 불가**한 **개인·비영리 목적**의 제한적 사용 등으로 해석되는 서술

공개 자료에 따르면 Gogolook은 **ASI(Anti-Scam Intelligence)** 등 **B2B용 API·데이터 피드**를 제안할 수 있음 → **사업적으로 허가된 경로(GREEN 후보)**는 **계약 상 존재 가능**.

| 항목 | 판정 |
|------|------|
| robots.txt (Whoscall 도메인) | **UNKNOWN** |
| ToS | 무단 자동화·역설계·무단 재배포 **강하게 금지** → 무허가 경쟁 피드 = **RED** |
| 공식 API/파트너 | 문헌상 존재 가능 → **계약 시 GREEN** |
| **종합** | **무허가 Competitor Feed 통합: RED**. **공식 영업·법무 채널(ASI 등)** 통해서만 **GREEN** 검토 |

## 5. 아키텍처·제품 권고

1. **§30-4-4**의 “**라이선스·이용약관 사전 검토 필수**”에 따라, 본 WO 범위에서는 **어느 경쟁사 대량 피드도 GREEN으로 확정하지 않았다**.
2. **즉시 구현 가능한 안전 경로**: **Gogolook 등 공식 상업 API·OEM/파트너 계약**이 명문화된 후에만 **GREEN**으로 재분류·설계.
3. **한국 3사**(더콜·후후·뭐야이번호): **공식 데이터 라이선스 또는 서면 허가** 없이 **벌크 크롤·미러링·재판매 목적 캐시**는 하지 않는 것이 안전하다.
4. **robots.txt만 GREEN으로 해석하지 말 것** — 저작권·부정경쟁방지·계약법상 책임은 별개다.

## 6. 증적·재현 (기술)

- **더콜 robots.txt**: Windows `curl.exe` + 브라우저 UA, 응답 본문 `User-agent: *` / `Disallow:` 빈 값 확인.
- **뭐야이번호 약관**: `http://idc-kr-notice.whycall.net/WHO2/agree/agree1.htm` 텍스트 확본 (제11조 인용).
- **Whoscall 약관**: `whoscall.com` 계열 이용약관 페이지에서 발췌된 번역·원문 혼합 텍스트.
- **Whoscall robots**: `whoscall.com/robots.txt` 등 — 본 세션에서 **안정적 텍스트 robots 미확보**.
